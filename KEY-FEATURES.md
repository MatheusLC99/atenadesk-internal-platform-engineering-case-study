# Key Features

## Clients
- standardized client profiles
- required fields + validation
- search, filters, pagination
- dynamic custom fields (admin-defined)

## Requests (Tickets)
- created and linked to a client
- department assignment (support / engineering)
- Kanban-style lifecycle:
  - Requested → In Review → In Progress → Done
- attachments for context and reproducibility

## Knowledge Base
- internal wiki for procedures, tips, and training
- global or client-specific articles
- attachments (PDF/images/zip)
- structured fields for context and usage guidance

## Admin
- user management (create/edit/delete)
- custom field management
- audit log viewer (who did what and when)
- legacy integration access (restricted)
