# UX Flows

AtenaDesk is designed for fast internal usage with clear navigation and minimal friction.

## Global Navigation
- Collapsible sidebar (mobile-friendly)
- Modules:
  - Dashboard
  - Clients
  - Requests
  - Knowledge Base
  - Admin (admin-only)

## Primary Journeys
1) Login → Dashboard
2) Create client → Create request → Track status (Kanban)
3) Search knowledge base → Open article → Download attachment
4) Admin: create user → assign role → audit actions
